      About VisioCirkit

      VisioCirkit is forked from the original CircuiTikZ-Designer and operates with identical behavior. The original project was part of the Circuit2TikZ-Project started as a research project at the Institute of Electronics Engineering at the Friedrich-Alexander-Universität Erlangen-Nürnberg.

      If you find bugs or have suggestions, please feel free to contact us, preferably on GitHub.

      VisioCirkit GitHub

      Tab management

      VisioCirkit (forked from CircuiTikZ-Designer with identical behavior) can manage a different canvas for each tab to allow multiple drawings at the same time.

      Help

      General

      • We use TikZ code as the single source of truth, so we don't use JSON to save circuits.
      • Every new browser tab is a new canvas ready for drawing. You can manage the browser tabs in the settings menu.
      • Closing/Reloading the tab will not erase the drawn circuit. Reopening the tab will reload the circuit.
      • Closing/Reloading the tab will erase the editing history (Undo and redo will be lost).
      • Reloading the tab can sometimes be useful if you encounter bugs.
